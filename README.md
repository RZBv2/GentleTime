# ✦ GentleTime ✦

A cozy, kawaii-aesthetic activity timer to track your day with gentle reminders. 💜

## What is GentleTime?

GentleTime is a soft, pastel-colored timer application built with Python's `tkinter`. It helps you:
- ⏱️ Set and track time blocks for different activities
- 📝 Log what you've been doing throughout the day  
- 💾 Remember your name (it saves it for next time!)
- 🎨 Enjoy a calming purple & blue gradient background
- 🌿 Feel supported by cute little animated elements

## Aesthetic Features

- **Gradient background**: Soft periwinkle → lilac → pale lavender vibes
- **Cute scenery**: Animated grass, flowers, and clouds ✨
- **Kawaii-friendly**: Designed to make productivity feel cozy, not stressful

## Getting Started

### Requirements
- Python 3.x
- `tkinter` (usually included with Python)

### Run It

```bash
python gentletime.py
```

When you first run it, GentleTime will ask for your name. It remembers! 💫

## Files Created

GentleTime will automatically create:
- `gentletime_user.txt` — Saves your name
- `gentletime_history.txt` — Logs your activities with timestamps

## Features

- ⏲️ Set custom timers for any activity
- 📊 Track activity history with timestamps
- 💬 Personalised greetings (it knows your name!)
- 🎨 Fully customizable kawaii aesthetic
